# Roadmap

---

## MVP — المرحلة الأولى
**الهدف:** نظام شغال كامل

- Auth + Registration + Invite System
- Projects + Sections + Tasks
- Files + Versioning
- Project Brain + Memory Engine
- AI Chat بسياق كامل
- Smart Dashboard
- Global Search
- Timeline
- Claude Analysis + Task Generation
- WhatsApp Sharing (wa.me)
- Decision Log + Activity Log
- PWA

---

## V2 — التوسع
**الهدف:** نظام احترافي للفريق

- Push Notifications
- Bug Tracker مستقل
- Approval Workflow متعدد الأشخاص
- HTML Preview داخلي
- Version Comparison (Diff بين نسختين)
- Templates جاهزة (HTML Review، UI Review...)
- AI Memory كاملة (Diff بين نسخ الكود)
- WhatsApp Business API الرسمي
- تصدير PDF للتقارير

---

## V3 — المنتج
**الهدف:** منتج قابل للبيع

- تطبيق APK
- SaaS (Multi-tenant — كل عميل له workspace مستقل)
- Billing System
- Custom Domain لكل عميل
- Advanced Analytics
- Integrations (GitHub، Slack، Jira)
